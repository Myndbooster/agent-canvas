## [1.0.0-alpha.2] - 2025-05-11

### Added

- Initial npm package release of `boostersdev`
- CLI entry point (`npx boostersdev`) to run full stack locally
- Library build mode with component barrel exports
- Subpath exports for modular imports:
  - `boostersdev/browser`
  - `boostersdev/conversation`
  - `boostersdev/files`
  - `boostersdev/settings`
  - `boostersdev/sidebar`
  - `boostersdev/terminal`
  - `boostersdev/i18n`
- TypeScript type declarations
- GitHub Actions workflow for automated npm publishing (OIDC trusted publishing)
